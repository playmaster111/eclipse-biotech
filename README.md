Welcome to the **Eclipse Biotech Platform**, a high-fidelity pharmacological research environment and clinical databank. This system is designed for advanced synthesis analysis and kinetic profiling of biochemical compounds.

## 🌟 Key Features

*   **Neural Link Boot Sequence**: Immersive, global system initialization sequence with real-time HUD overlays.
*   **Interactive 3D Holograms**: High-fidelity 3D models of pills, vials, and molecules with interactive rotation and laser scan effects.
*   **Cortex AI Engine**: Context-aware AI assistant with deep-dive clinical reporting and thought logs.
*   **High-Fidelity Databank**: Comprehensive database of AAS, SARMs, Peptides, and Nootropics with detailed aesthetic profiles and muscle-specific targets.
*   **Multi-Language Interface**: Full support for English, German, Italian, and Spanish.
*   **Cycle Architect**: Advanced tool for designing and analyzing research protocols.

## 🛠 Tech Stack

*   **Core**: HTML5 / CSS3 / Vanilla JavaScript
*   **3D Engine**: Three.js
*   **Styling**: Modern CSS with Glitch Effects, Chromatic Aberration, and Blur-to-Focus transitions.
*   **Data**: JSON-based pharmacological repository (`data.js`).

## 🚀 Quick Start

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/eclipse-biotech.git
    ```
2.  Open `index.html` in a modern browser (Edge/Chrome recommended).
3.  Press `TAB` to toggle the navigation sidebar.
